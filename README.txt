Personal project n.1 - Japan Estate [WIP]-

- DESCRIPTION -
This project is about a web app that simulates the purchasing of houses in Japan.
Users can register and login in the website in order to add houses to their basket and purchase them. The catalog has a Filters-bar to facilitate the searching


- TECHNOLOGIES INVOLVED -

BACK-END:
- Java
- Spring
- Hibernate

FRONT-END:
- Html / Css / JavaScript
- Thymeleaf

DATABASE:
- MySQL
